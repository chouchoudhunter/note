## 1.bind（）
{: id="20201027230918-8bjvc2r"}

```javascript
window.a=3
function test(){
	console.log(this.a)
}
test.bind({a:10})()
//输出10而不是3，bind让函数动态绑定了一个对象，this指向这个对象
```
