## 2.数组

```javascript
data() {
    return {
      test:[{ val: 0 }]
}
```
{: id="20201215114522-gqlfape"}

```xml
<div v-for="(item,index) in test" :key="index">{{ item.val }}:{{ item.name }}</div>
<a-button @click="clickTest()">Test</a-button>
```
